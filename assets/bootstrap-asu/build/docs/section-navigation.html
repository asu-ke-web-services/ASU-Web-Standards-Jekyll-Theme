15 Navigation #navigation

There are many different navigation options available in this theme.

Each one should be used appropriately to properly convey information in a way that users will understand.

15.1 Alternative Tabs #navigation.alternative-tabs

Used in conjunction with .nav and .nav-tabs

Example
  • Record A
  • Record B
  • Record C
<ul class="nav nav-tabs alterna-tabs">
   <li class="active"><a href="#">Record A</a></li>
   <li><a href="#">Record B</a></li>
   <li><a href="#">Record C</a></li>
</ul>

15.2 Justified Navs #navigation.justified-navs

Can be used with .nav-tabs, or .alterna-tabs

Example
  • Home
  • Profile
  • Messages
<ul class="nav nav-tabs nav-justified">
  <li role="presentation" class="active"><a href="#">Home</a></li>
  <li role="presentation"><a href="#">Profile</a></li>
  <li role="presentation"><a href="#">Messages</a></li>
</ul>

15.3 Justified Alternative Nav #navigation.justified-navs-alternative

Example
  • Home

<ul class="nav nav-tabs alterna-tabs nav-justified">

15.6 Navbars #navigation.navbar

Just like in Bootstrap, but styled differently.

Note that .navbar-brand is hidden on large screens.

Example
  • Link
  • Dropdown
    • Action
    • Another action
    • Something else here
    • Separated link
<nav class="navbar navbar-default">
  <div class="container-fluid">
    <ul class="nav navbar-nav navbar-right">
      <li><a href="#">Link</a></li>
      <li class="dropdown">
        <a href="#" class="dropdown-toggle" data-toggle="dropdown" role="button" aria-haspopup="true" aria-expanded="false">Dropdown <span class="caret"></span></a>
        <ul class="dropdown-menu">
          <li><a href="#">Action</a></li>
          <li><a href="#">Another action</a></li>
          <li><a href="#">Something else here</a></li>
          <li role="separator" class="divider"></li>
          <li><a href="#">Separated link</a></li>
        </ul>
      </li>
    </ul>
  </div>
</nav>

15.7 Sidebar #navigation.sidebar

Can be used with .affix. If you do want the sidebar to be affixed, you need to add id="sidebarNav" and class="affix". You should also use the push and pull variants of the col-*-* classes.

Here is an example of the columns, note that even though the sidebar comes first in the markup, it will be on the right hand side of the page:

<div class="col-sm-4 col-sm-push-8">[sidebar goes here]</div>
<div class="col-sm-8 col-sm-pull-4">[content goes here]</div>

15.8 Web Standards Navbar #navigation.web-standards

Just like in Bootstrap, but styled differently, uses navbar. Dropdown menus must include caret icon as styled here.

Additionally, a Navbar Button is available for special uses. All Alternative Button Colors are available for styling the Navbar Button.

Example
Global Sustainability
  • Home
  • Simple Dropdown
  • Super Menu
  • Tabbed Menu
  • Navbar Button 
<nav class="navbar navbar-ws affix-top" role="navigation">
    <div class="container-fluid">
        <div class="navbar-header">
            <button type="button" class="navbar-toggle" data-toggle="collapse" data-target="#ws-navbar-collapse-1" style="display: inline-block;">
                <span class="sr-only">Toggle navigation</span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
            </button>
            <a class="navbar-brand" href="#">Global Sustainability</a>
        </div>
        <div id="ws-navbar-collapse-1" class="collapse navbar-collapse">
            <ul id="menu-header-menu" class="nav navbar-nav">
                <li><a href="#" id="home-icon-main-nav"><span class="fa fa-home hidden-xs hidden-sm" aria-hidden="true"></span><span class="hidden-md hidden-lg">Home</span></a>
                </li>
                <li class="menu-item"><a title="Simple Dropdown" href="#" id="nav-item_outreach">Simple Dropdown <span class="caret"></span></a>
                </li>
                <li class="menu-item"><a title="Super Menu" href="#" id="nav-item_research">Super Menu <span class="caret"></span></a>
                </li>
                <li class="menu-item"><a title="Tabbed Menu" href="#" id="nav-item_solutions">Tabbed Menu</a>
                </li>
                <li class="menu-item menu-btn-navbar"><a title="Navbar Button" href="#" id="nav-item_button" class="btn-navbar btn btn-gold">Navbar Button</a>
                </li>
            </ul>
        </div>
    </div>
    <!-- /.navbar-collapse -->
</nav>

15.8.1 Navbar States #navigation.web-standards.states

There are 4 main states that need to be consistent within the navbar:

Example
  • Normal
  • Active
  • Hover
  • Disabled 
<nav class="navbar navbar-ws affix-top" role="navigation">
    <div class="container-fluid">
        <div class="collapse navbar-collapse">
            <ul class="nav navbar-nav">
                <li><a href="#">Normal</a>
                </li>
                <li class="active"><a href="#">Active</a>
                </li>
                <li class="hover"><a href="#">Hover</a>
                </li>
                <li class="disabled"><a href="#">Disabled</a>
                </li>
            </ul>
        </div>
    </div>
</nav>

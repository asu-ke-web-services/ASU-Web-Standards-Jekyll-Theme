5 Calendars #calendars

A special web standards widget. Use the .calendarPopover class to make calendars have a popover with a download widget.

Example
Sun 1 Jan
Tue 8 Mar
Wed 27 Jun
Fri 30 Oct 
<div class="calendar-group">
  <div class="calendar-date calendarPopover" data-original-title="" title="">
    <time datetime="2014-01-01">
      <span class="weekday">Sun</span>
      <span class="date">1</span>
      <span class="month">Jan</span>
    </time>
  </div>
  <div class="calendar-date calendarPopover" data-original-title="" title="">
    <time datetime="2014-03-08">
      <span class="weekday">Tue</span>
      <span class="date">8</span>
      <span class="month">Mar</span>
    </time>
  </div>
  <div class="calendar-date calendarPopover" data-original-title="" title="">
    <time datetime="2014-06-27">
      <span class="weekday">Wed</span>
      <span class="date">27</span>
      <span class="month">Jun</span>
    </time>
  </div>
  <div class="calendar-date calendarPopover last" data-original-title="" title="">
    <time datetime="2014-10-30">
      <span class="weekday">Fri</span>
      <span class="date">30</span>
      <span class="month">Oct</span>
    </time>
  </div>
</div>

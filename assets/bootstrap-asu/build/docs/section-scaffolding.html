21.3 Headings Using Modifiers #scaffolding.headings-using-modifiers

You can also style any text to look like a heading by using classes instead of markup. This should be refrained from though since it will not be accessible by screen readers.

21.4 Images #scaffolding.images

Optional CSS Modifiers

  • .img-bordered — 
    Add a 1 pixel border to an image
  • .img-border — 
    An alias of .img-bordered
  • .img-center — 
    Center an image
  • .img-100 — 
    Force an image to 100% width

21.5.1 External Links #scaffolding.links.external

These are used by the navigation. If a link is in a list, you can add .external-link to the <li> and the link will have an external icon on it.

21.9 Spacing #scaffolding.spacing

Optional CSS Modifiers

  • .margin-top-0 — 
    an alias for .space-top-0
  • .space-top-0 — 
    0px margin top
  • .space-top-sm — 
    10px margin top
  • .space-top-md — 
    20px margin top
  • .space-top-lg — 
    30px margin top
  • .space-top-xl — 
    40px margin top
  • .space-bot-0 — 
    0px margin bottom
  • .space-bot-sm — 
    10px margin bottom
  • .space-bot-md — 
    20px margin bottom
  • .space-bot-lg — 
    30px margin bottom
  • .space-bot-xl — 
    40px margin bottom
  • .mr-20 — 
    margin right 20px
  • .pad-top-0 — 
    0px padding top
  • .pad-top-sm — 
    10px padding top
  • .pad-top-md — 
    20px padding top
  • .pad-top-lg — 
    30px padding top
  • .pad-top-xl — 
    40px padding top
  • .pad-bot-0 — 
    0px padding bottom
  • .pad-bot-sm — 
    10px padding bottom
  • .pad-bot-md — 
    20px padding bottom
  • .pad-bot-lg — 
    30px padding bottom
  • .pad-bot-xl — 
    40px padding bottom
  • .pad-zero — 
    0 padding all around
  • .pad-md — 
    20px padding all around

There are a variety of spacing modifiers to help you tweak the spacing between your elements, while still maintaining a standardized layout

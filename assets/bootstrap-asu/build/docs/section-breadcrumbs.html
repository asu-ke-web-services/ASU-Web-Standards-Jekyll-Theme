3 Breadcrumbs #breadcrumbs

Breadcrumbs are supported, but when using WordPress, do not generate them by hand.

Example
  1. Home
  2. Library
  3. Data
<ol class="breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Library</a></li>
  <li class="active">Data</li>
</ol>

3.1 ASU Breadcrumbs #breadcrumbs.asu

For use with the Yoast SEO plugin

Example
Home / Education / ASU Sustainability Education 
<div id="breadcrumbs" class="breadcrumb">
  <span xmlns:v="http://rdf.data-vocabulary.org/#">
    <span typeof="v:Breadcrumb">
      <a href="https://wp.sustainability.dev.gios.asu.edu" rel="v:url" property="v:title">Home</a>
      /
      <span rel="v:child" typeof="v:Breadcrumb">
        <a href="https://wp.sustainability.dev.gios.asu.edu/education/" rel="v:url" property="v:title">Education</a>
        /
        <span class="breadcrumb_last">ASU Sustainability Education</span>
      </span>
    </span>
  </span>
</div>

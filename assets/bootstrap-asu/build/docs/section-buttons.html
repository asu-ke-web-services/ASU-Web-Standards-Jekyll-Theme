4 Buttons #buttons

Use the button classes on <a>, <button>, or <input>.

Example
Link 
<div>
 <a class="btn btn-default" href="#" role="button">Link</a>
 <button class="btn btn-default" type="submit">Button</button>
 <input class="btn btn-default" type="button" value="Input">
 <input class="btn btn-default" type="submit" value="Submit">
</div>

4.1 Alternative Buttons #buttons.alternative-buttons

Use any of the available button classes to quickly create a styled button.

Examples
Default styling
.btn-default
Default button
.btn-primary
ASU maroon color
.btn-secondary
Gray button
.btn-gold
Gold button
.btn-degree-programs
Only for use with degree programs. (Formally .btn-blue)
.btn-success
Success action color
.btn-info
Information action color
.btn-warning
Warning color
.btn-danger
Danger color
.btn-link
A button styled as a link 
<button type="button" class="btn [modifier class]">[modifier class]</button>

4.2 Button Block #buttons.block

Create block level buttons—those that span the full width of a parent— by adding .btn-block.

Example 
<button type="button" class="btn btn-primary btn-lg btn-block">Block level button</button>

4.3 Button Sizes #buttons.sizes

Fancy larger or smaller buttons? Add .btn-lg, .btn-sm, or .btn-xs for additional sizes.

Examples
Default styling

.btn-lg
Large button

.btn-sm
Small button

.btn-xs
Extra small button 

<p>
  <button type="button" class="btn btn-primary [modifier class]">Button Text</button>
  <button type="button" class="btn btn-default [modifier class]">Button Text</button>
</p>
